# JavaScript Projects

This repository contains multiple small projects built using HTML, CSS, and JavaScript.
Each project focuses on strengthening fundamental JavaScript concepts through practical implementation.

These projects are ideal for:

- Beginners learning JavaScript

- Practicing DOM manipulation

- Understanding event handling

- Improving problem-solving skills

## 🚀 Technologies Used

- HTML5 – Structure

- CSS3 – Styling

- JavaScript

## 📂 Projects Included

- BMI Generator

- Digital Clock

- Number Guessing Game

- KeyBoard

- Color Generator

- Background Color Switcher


## 🎯 What You Will Learn

- DOM Selection & Manipulation

- Event Listeners

- Functions & Conditional Logic

- Random Number Generation

- Array Methods

## ▶️ How to Run

- Clone or download the repository.

- Open any project folder.

- Open the index.html file in your browser.

- Start interacting with the project 🎉
